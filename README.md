# Glorious Gourd Game
This is an awesome console game about Gourds, Balls and really cool Weapons.

## Install
1. Download zip
2. Unzip
3. Rename 'save_data.example.json' to 'save_data.json'
4. Download Pygame if you don't have it (pip install pygame in console)
5. Run 'Main.py' (Through VSCode or whatever you use, there's no .exe right now)

#### IMPORTANT CURRENTLY NEEDS PYTHON 3.12

## Updating
If you wish to update the game without losing your save you can simply drag in your previous 'save_data.json' file into the new folder (you're gonna have to download the new zip for the game and unzip it just like when you first installed the game). Don't rename the 'save_data.example.json' this time since you already have a save file from before. Once this is done you can safely delete the old version of the game and start playing on the new one. Happy gourding!

## Cheating
The game is originally based around being as annoying as possible and evolved into what it is today. Today's version is obviously way more playable and can at times even be fun :)! While I would recommend you to play through most of it, if you REALLY find it quite boring to grind through the game and learn all the different stats and concepts, you can always change your current stats in the save_data.json file, very easily. You can also read all the stats for the weapons in the item_data.json file though it is HEAVILY discouraged.
